HTML



태그 종류

TAG: html에서 사용하는 컴퓨터 언어

u : 밑줄 태그
h 숫자 : 제목 태그
br : 줄 바꿈 태그
p : 문단 나누기 태그 - br태그보다 정보로서 가치있음
img 태그: 사진 태그이며 width(사진크기), src(사진파일명) 등의 속성을 부여해 줄 수 있다.
li, ul, ol : 부모자식 태그 관계이며 li는 목차태그, ul은 li의 부모 태그이다. 또한 ol태그는 자동 넘버링 태그이다.
meta utf: 정보를 utf로 저장하게 해줘서 출력 보정
head: 제목을 의미해주는 정보
body: 내용 의미 정보
html: html문서를 의미한다 a: 링크를 걸어주는 태그 눈물의 씨앗 웃음의 꽃
